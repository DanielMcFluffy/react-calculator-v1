# Simple Calculator App Using React Typescript

## Overview

A short first mini project I did using React while learning to write type-safe code with Typescript. 

![Alt Text](./assets/calculator.png)

## Features

1. Responsive design that accomdates smaller window sizes
2. Calculation sequence adheres to mathematical order of operations
3. Usage of interfaces/generics/unions/etc for better type-safe code

## Installation

1. Clone the repository: `git clone https://github.com/username/repository.git`
2. Navigate to the project directory: `cd repository`
3. Install dependencies: `npm install`
4. Run project: `npm run dev`
